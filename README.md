# PokéBloom - Frontend

## Descripción del proyecto

PokéBloom es una Pokédex web con un estilo pastel y amigable. Permite explorar el catálogo de Pokémon, buscar, ver la ficha completa de cada uno, filtrar la colección, guardar favoritos, armar equipos y ver su sinergia, consultar estadísticas de la comunidad, y contiene funciones propias como un modo coleccionista con insignias, un minijuego de adivinar Pokémon, la opción de compartir fichas o equipos, un diario personal de notas, y un Pokémon destacado cada día.

Este repositorio contiene el frontend de la aplicación, encargado de la interfaz y de la experiencia de usuario.
